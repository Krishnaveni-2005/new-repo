Linux DF Command

5. df

Description

The df command in Linux is used to display information about the available and used disk space on file systems. It's a useful tool for monitoring disk usage and ensuring your system has enough space for operations.

Purpose and Usage

The df command provides a snapshot of how much disk space is being used and how much is available on each mounted file system. This is helpful for managing disk space, especially when dealing with multiple partitions or storage devices.

Syntax:

df [options]

Examples

To display disk space usage in a human-readable format (e.g., GB, MB):

df -h

Output might include:

Filesystem      Size  Used Avail Use% Mounted on
/dev/sda1       100G   20G   75G  21% /
/dev/sdb1       200G  120G   80G  60% /mnt/data

Screenshot

Additional Explanation

The df command can be used with several options to customize the output. Some useful options include:

  • -h: Human-readable output (e.g., MB, GB)
  • -T: Display the type of file system
  • -a: Include file systems with 0 blocks

The output from the df command typically includes the file system, size, used space, available space, percentage used, and the mount point for each file system. This information helps you monitor disk usage and manage storage effectively.
