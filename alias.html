Linux Alias Command

3. alias

Description

The alias command in Linux is used to create custom shortcuts for long or frequently used commands. Aliases help improve productivity by saving time when typing long command strings.

Purpose and Usage

The alias command is used to define a new name (alias) for a command or series of commands. This can be useful for simplifying complex commands or for creating personalized command shortcuts that suit your workflow.

Syntax:

alias [name]='[command]'

Examples

To create an alias for the ls -l command to make it shorter:

alias ll='ls -l'

To view the list of current aliases:

alias

Screenshot

Additional Explanation

The alias command is helpful for creating custom command shortcuts to speed up tasks. For example, you could create aliases for commonly used commands, like:

alias update='sudo apt-get update && sudo apt-get upgrade'

Aliases are temporary and only last for the duration of the session. To make them permanent, you need to add them to your shell's configuration file (e.g., .bashrc or .zshrc).
