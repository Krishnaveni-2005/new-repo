Linux DU Command

6. du

Description

The du command in Linux is used to estimate and display the disk space used by files and directories. It's a useful tool for understanding disk usage on your system, especially when trying to identify large files or directories.

Purpose and Usage

The du command provides a way to check the size of files and directories on your system, either for a specific directory or for the entire file system. This command helps you manage disk space by identifying which files or directories are consuming the most storage.

Syntax:

du [options] [directory]

Examples

To display the disk usage of the current directory:

du

To display the disk usage of the current directory in a human-readable format (e.g., KB, MB):

du -h

Output might include:

4.0K    ./dir1
2.3M    ./dir2
1.5G    ./dir3

Additional Explanation

The du command can be used with several options to customize the output. Some useful options include:

  • -h: Human-readable output (e.g., KB, MB, GB)
  • -s: Display only the total size for the specified directory
  • -a: Display the disk usage for all files, not just directories
  • -c: Show a grand total at the end of the output

The du command is particularly useful for monitoring and managing disk usage, especially in environments with limited storage. You can combine it with other commands like grep to find specific files or directories that are using large amounts of space.
